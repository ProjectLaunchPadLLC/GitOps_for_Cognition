# Inference Cortex

## Definition

The Inference Cortex is the external reasoning layer composed of Large Language Models (LLMs) used as bounded cognitive operators within the system.

---

## Core Purpose

- Perform reasoning tasks within DAG nodes
- Transform inputs into structured outputs
- Execute bounded cognitive functions

---

## Key Principle

> The model does not decide the workflow. It only executes transformations.

---

## Novelty

Traditional systems:
- LLM = full agent

This system:
- LLM = stateless function inside cognitive pipeline

---

## Impact

- Prevents uncontrolled autonomy
- Enables model swapping (GPT, Gemini, local)
- Improves modularity and cost optimization
- Enables multi-model routing

---

## Value

- Vendor independence
- Cost-efficient inference routing
- High reliability cognitive systems
- Scalable reasoning architecture
